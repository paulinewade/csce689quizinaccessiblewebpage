Inaccessible Webpage Quiz

Overview

This webpage has several accessibility errors. You should have been exposed to all of the issues found here.

Allowed Resources

1. You can use any course and online resources as long as you work alone. 2. Accessibility trumps other aspects of the presentation. So, your accessible version's presentation might be different from the original, but it should be structurally and semantically equivalent. 3. Hint: The activities we did in class and material from the textbooks is sufficient reference material. To find this information on Canvas, Access TAMU Canvas.

What to Do

  1. Download a copy of the page to edit its HTML.

  2. You can generally do this by viewing the page in a browser and choosing to "save page as" (e.g., use Ctrl+S).

  3. Resolve the accessibility issues.

  4. There are eight accessibility issues.

  5. Rubric:

  6. You start with 2 points for attempting to resolve the issues regardless of success.

  7. For each issue:

    1. You get one point for completely resolving the issue.

    2. You get half a point for partially resolving the issue. For example, fixing it in one place but not in other places. Not all issues can be partially resolved.

  8. Total of 10 points.

  9. Submit the updated HTML file.

  10. You may also submit CSS files if you do the challenge, but you can also use internal CSS.

  11. As a challenge, use CSS to make the accessible HTML render more like the original.
